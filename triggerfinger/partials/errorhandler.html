Error Handler

A module employing the Visitor pattern used to handle faulty workflows. It consists of two classes:
  • ErrorHandlingVisitor - Class implementing the Visitor pattern. Implemented method is for handling faulty workflows. It simply re-runs it.
  • ErrorHandlingSchedule - Class which periodically runs the ErrorHandlingVisitor handling on all the failures that are being added to the system failures lists. It checks, whether the limit of re-runs has not been exceeded and re-runs the flow.
